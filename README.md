# Proyecto Visión Artificial - Fase 1

Este proyecto implementa las funciones básicas de captura y procesamiento de vídeo en tiempo real usando OpenCV.

## Requisitos de Hardware
* **Ordenador con Webcam integrada** o cámara USB externa.
* Procesador capaz de manejar flujo de vídeo en tiempo real.

## Requisitos de Software
* **Python 3.10+**
* Librerías listadas en `requirements.txt`.

## Funcionalidades incluidas:
1. **Captura inicial:** Guarda una foto automática al arrancar.
2. **Modo LIVE:** Streaming de vídeo con cierre mediante tecla 'q'.
3. **Procesado de imagen:** Dibujo de círculo central y reescalado de ventana.
4. **Rendimiento:** Cálculo de FPS y tiempo de refresco en milisegundos (ms) visibles en pantalla.
